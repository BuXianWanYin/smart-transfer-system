OpenSkills 技能说明

基于 Anthropic 官方 Skills，在 Cursor 中通过 AGENTS.md 使用。调用方式：npx openskills read <技能名>

论文格式：广西外国语学院本科毕设论文模板的全部格式已写入 Cursor 规则，可通过 @.cursor/rules/thesis/thesis-format.md 读取；或在与「论文格式」「论文模板」「WPS 格式化」相关对话时由规则自动提供。

序号 技能名 说明
1 algorithmic-art 使用 p5.js 创建算法艺术，支持随机种子与交互式参数探索。适用于：用代码做艺术、生成艺术、算法艺术、流场、粒子系统。会创作原创算法艺术，避免抄袭他人作品。
2 brand-guidelines 将 Anthropic 官方品牌色与字体应用到各类产出物，使其具有 Anthropic 的视觉风格。适用于：品牌色、风格指南、视觉排版或公司设计规范相关需求。
3 canvas-design 基于设计哲学在 .png 和 .pdf 中创建视觉作品。适用于：海报、艺术作品、设计稿或其他静态视觉。创作原创设计，不复制现有艺术家作品。
4 doc-coauthoring 引导用户完成结构化文档协作流程。适用于：写文档、提案、技术规格、决策文档等。帮助高效传递上下文、迭代润色并验证文档对读者的效果。触发词：写文档、写提案、起草规格等。
5 docx 完整的 Word 文档创建、编辑与分析，支持修订、批注、格式保留与文本提取。适用于：新建文档、修改内容、处理修订、添加批注等与 .docx 相关的任务。
6 frontend-design 创建有辨识度、可上线的前端界面，保证较高设计质量。适用于：Web 组件、页面、海报或应用（如网站、落地页、仪表盘、React 组件、HTML/CSS 布局，或美化任意 Web UI）。生成有创意、精致的代码与 UI，避免通用 AI 风格。
7 internal-comms 提供撰写各类内部沟通内容的资源与公司常用格式。适用于：状态报告、领导层更新、第三方更新、公司通讯、FAQ、事故报告、项目更新等内部沟通。
8 mcp-builder 创建高质量 MCP（Model Context Protocol）服务器的指南，让 LLM 通过设计良好的工具与外部服务交互。适用于：用 Python（FastMCP）或 Node/TypeScript（MCP SDK）搭建 MCP 服务器、对接外部 API 或服务。
9 pdf PDF 处理工具：提取文本与表格、创建新 PDF、合并/拆分文档、处理表单。适用于：填写 PDF 表单，或以编程方式批量处理、生成或分析 PDF。
10 pptx 演示文稿的创建、编辑与分析。适用于：新建演示、修改内容、调整版式、添加批注或演讲者备注等与 .pptx 相关的任务。
11 skill-creator 创建有效技能的指南。适用于：想新建或更新一个技能，以扩展 Claude 在专业知识、工作流或工具集成方面的能力。
12 slack-gif-creator 为 Slack 制作动图的知识与工具，包含约束、校验与动画思路。适用于：用户要求“为 Slack 做一个 X 在做 Y 的 GIF”等场景。
13 template 占位技能，需替换为具体技能描述及使用场景。
14 theme-factory 为产出物应用主题样式的工具，可作用于幻灯片、文档、报告、HTML 落地页等。提供 10 种预设主题（颜色/字体），也可即时生成新主题。
15 web-artifacts-builder 使用现代前端技术（React、Tailwind CSS、shadcn/ui）创建复杂的多组件 HTML 产出物。适用于需要状态管理、路由或 shadcn/ui 组件的复杂产出物，不适用于简单单文件 HTML/JSX。
16 webapp-testing 使用 Playwright 与本地 Web 应用交互并测试。支持验证前端功能、调试 UI 行为、截屏、查看浏览器日志。
17 xlsx 电子表格的创建、编辑与分析，支持公式、格式、数据分析与图表。适用于：新建带公式与格式的表格、读取或分析数据、在保留公式的前提下修改表格、在表格内做数据分析与可视化、重算公式等。支持 .xlsx、.xlsm、.csv、.tsv 等。
— 论文格式（本地规则） 广西外国语学院本科毕设论文格式模板全文规范（封面、目录、摘要、正文、参考文献、致谢、题注、页眉等字号/行距/对齐）。读取方式：在 Cursor 中用 @.cursor/rules/thesis/thesis-format.md 引用，或在与「论文格式」「论文模板」「WPS 格式化」相关对话时由规则自动提供。WPS 宏见 WpsFormatThesis/GXWGY_FormatThesis_WPS_v1.js。
